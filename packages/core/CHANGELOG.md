# lovely-alert

## 0.1.0

### Minor Changes

- c6c5fe5: Initial public release (0.1.0).

  - **lovely-alert** — modern, accessible, themeable alert / modal / toast core (vanilla TS, zero runtime deps): light/dark/auto theming, full WAI-ARIA, 18 input types with sync/async validation, stacking toasts, timers & progress steps, fluent builder, effects (confetti/sound), runtime theme builder, and i18n (en + vi).
  - **@lovely-alert/react** / **@lovely-alert/vue** — framework bindings (hook / plugin + composable).
  - **@lovely-alert/catalog** — shared example catalog (single source of truth for docs/MCP/tests).
  - **lovely-alert-mcp** — MCP server (stdio + HTTP) exposing the catalog, code generation, API reference and theme tooling.
